Detailed Description

Класс, созданный для работы с датами

В данном классе переопределены операции сравнения и вывода. В том числе реализованы функции для получения значения полей класса. Поля класса следующие:

  • day - день даты
  • month - номер месяца
  • year - номер года

При создании объекта класса, проверяется корректность введенной даты, и в случае ее некорректности все поля класса принимают значение 0
